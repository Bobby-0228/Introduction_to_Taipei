## Project Overview

**Introduction to Taipei** is a modern, dark-themed travel guide that helps users quickly explore the essential sights, transportation information, and cultural highlights of Taipei.

The website includes:

- A landing page

- A main page with hero image sliders

- Multiple content sections (Sights, Essentials, Orientation, Transport)

- Dedicated sub-pages for each attraction
